Tipificação:
Estática(forte)-obriga definir o tipo no inicio da declaração vs Dinâmica(fraco)
Primitivo vs Composto

Tipos; textuais, numeral, lógico, objeto
Numero: byte, short, int, long, float, double.
Textual: char, String
Lógico: Boolean

boas práticas:
